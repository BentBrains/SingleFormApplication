Manage Form Example

This an example of won to manage form SUBMIT.

As the browser try to submit the form to a server, we need to intercept this action and send it to our VBA function.

If you use the xhvBootstrapSubmitButtonComponent, (pretty long name...) all of this is handle by the component.
If you don't need to validate anything via javascript, you can use a standard button instead of a Submit button like in the first example.

If you need to submit the form and validate in javascript, check the xhvBootstrapSubmitButtonComponent code to see how we do.
Basically, we trap the submit event in javascript, and then call the click event of a hidden button that contains the the event handler we want to call in VBA.
