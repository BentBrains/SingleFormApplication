Button example

This an example of re-usable component.

This button component is pretty useless by it self, but it shows you how to make a re-usable component.

Look into the HTML and VBA code how to implements properties for your components using interpolations and props dictionary.

HTML Button

This is a normal HTML button.

This button calls a VBA Sub that will execute a javascript function in the buttonExample component.

HandleView component

This is a HandleView component.

This button calls a VBA Sub in the component.

HandleView component

This is a HandleView component.

This button calls a VBA Sub that will execute a method in the parent component. For this button, the parent component is buttonExample component.

xhvBootstrapSubmitButtonComponent HandleView component

Now this is a usefull HandleView component !
